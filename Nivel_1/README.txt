📄Descripció - Enunciat de l'exercici

Ejercicio 1 y 2 del nivel 1 de MySQL (Estructura de datos).

💻 Tecnologies Utilitzades

Aquest projecte ha estat desenvolupat utilitzant les següents tecnologies i eines:

MySQL: Base de dades relacional per a l'emmagatzematge i manipulació de dades.

MySQL Workbench: Eina per a la modelització de bases de dades i execució de consultes SQL.

SQL: Llenguatge utilitzat per a la definició i manipulació de dades.

📋 Requisits

Per a poder executar aquest projecte, és necessari comptar amb els següents requisits:

MySQL Server (versió 8.0 o superior) instal·lat i configurat.

MySQL Workbench o una altra eina similar per a executar les consultes.

Coneixement bàsic de SQL, especialment sobre JOIN i manipulació de dades.

🛠️ Instal·lació

Segueix aquests passos per a instal·lar i preparar l'entorn local:

Instal·la MySQL Server i MySQL Workbench si encara no ho tens instal·lat.

Obre MySQL Workbench i connecta't al servidor MySQL.

Executa el fitxer .sql proporcionat per a crear i carregar la base de dades.



▶️ Execució

Per a executar les consultes SQL:

Obre MySQL Workbench i selecciona la base de dades correcta (USE nom_de_la_base_de_dades;).

Copia i enganxa les consultes SQL proporcionades en aquest exercici.

Executa les consultes i revisa els resultats obtinguts.
